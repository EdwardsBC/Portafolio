Edwards Barboza - Especializado en Sistemas de Información

Desarrollador con más de 3 años de experiencia creando soluciones tecnológicas completas mediante arquitecturas como MVC y Microservicios, tanto en entornos empresariales como en proyectos personales profesionalizados. Cuento con dominio de lenguajes como Python, Java y Visual Basic, así como sólida experiencia en bases de datos relacionales como MySQL, SQL Server y PostgreSQL.

He liderado proyectos en todo su ciclo de vida: desde el análisis de requerimientos y entrevistas con clientes, hasta la planificación, desarrollo, documentación y mantenimiento. Entre mis logros destaca la implementación de un sistema contable integral con soporte para Balance General, Estados Financieros, Kardex y registro de operaciones bursátiles, aplicando métodos avanzados como PEPS y Costo Promedio. Este sistema fue migrado exitosamente de PostgreSQL a MySQL, optimizando el rendimiento mediante herramientas como Navicat.

Como fundador de Golden Shulker Studios, he liderado un equipo multidisciplinario para la entrega de 9 proyectos personalizados para videojuegos, desarrollando plugins en Java (Spigot, Bukkit, Fabric), automatizaciones con bots en Discord (Python), y paneles administrativos web con Django y MySQL. Este entorno me ha permitido aplicar buenas prácticas de control de versiones (Git/GitHub), despliegue en servidores dedicados (Linux) y manejo eficiente de estructuras como YAML y JSON para configuración modular.

Mis habilidades incluyen desarrollo de APIs RESTful, procesamiento de datos con Pandas, automatización de reportes (Excel y PDF), así como diseño de interfaces gráficas con PySide6 y prototipado en Figma. Me apasiona mantenerme actualizado en tecnologías emergentes y aplicar ese conocimiento a desafíos reales.

Cuento con formación técnica en Desarrollo de Sistemas, certificaciones en JavaScript y SQL (SoloLearn y Alura Oracle) y nivel intermedio de inglés (B1 certificado por SENATI). Actualmente busco integrarme en equipos que valoren la colaboración, la innovación y el aprendizaje continuo, con énfasis en la calidad técnica, planificación ágil y la entrega de valor real en cada proyecto.

Además, desarrollo proyectos personales aplicando tecnologías web y videojuegos, como datapacks para Minecraft y páginas web para eventos competitivos. Estos retos complementan mi perfil técnico y reflejan mi capacidad de adaptación, creatividad y compromiso con la excelencia profesional.
