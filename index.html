Sobre Mí

Edwards Barboza

Desarrollador de Sistemas de Información e Informática

Como profesional con más de 3 años de experiencia en el desarrollo de sistemas de información e informática, me destaco por mi eficiencia, adaptabilidad y habilidades técnicas en la creación de bases de datos, programación en lenguajes como Visual Basic, Python y Java, así como en el manejo de sistemas de gestión de bases de datos como SQL Server 2014, MySQL y PostgreSQL. Además, he liderado equipos de trabajo, brindando soluciones efectivas y escalables para diversos proyectos.

Uno de mis logros más destacados fue la participación en el desarrollo e implementación de un sistema contable integral. Este sistema optimiza procesos clave como la gestión del Balance de Comprobación, Balance General, Estados de Resultados, Kardex y el registro de transacciones de compra y venta de acciones para múltiples clientes. Para ello, se aplicaron métodos de cálculo avanzados como PEPS (Primeras Entradas, Primeras Salidas) y Costo Promedio. Inicialmente, el sistema fue implementado con una base de datos en PostgreSQL y posteriormente migrado a MySQL utilizando herramientas como Navicat, garantizando una transición fluida y eficiente.

Además, soy fundador de *Golden Shulker Studios*, un estudio especializado en el desarrollo de eventos y plugins para videojuegos. En este rol, he liderado la gestión integral del proyecto, incluyendo la programación en Java utilizando Visual Studio Code y Maven para el desarrollo de plugins. También diseñé e implementé un bot para Discord utilizando Python, y desarrollé la página web del estudio con Django y MySQL, demostrando mi versatilidad y capacidad para abordar proyectos de diversa índole.

Mi pasión por la programación va de la mano con mi compromiso de mantenerme actualizado en las últimas tendencias tecnológicas. Me motiva enfrentar desafíos que me permitan aprender y crecer profesionalmente. Cuento con certificaciones en JavaScript y SQL de plataformas como SoloLearn y ALURA ORACLE, además de un título técnico en Desarrollo de Sistemas y un certificado de inglés básico.

Actualmente, busco integrarme en empresas que fomenten el desarrollo continuo de habilidades y que valoren la innovación y la colaboración. Mi objetivo es contribuir con mis conocimientos y compromiso en entornos que adopten metodologías ágiles y una planificación rigurosa para la gestión eficaz de proyectos. Aspiro a formar parte de organizaciones que promuevan el crecimiento profesional y ofrezcan oportunidades para participar en la concepción y ejecución de proyectos desafiantes y transformadores.

Además de mi experiencia técnica, disfruto explorando nuevas tecnologías y aplicándolas en proyectos personales, como la creación de datapacks para Minecraft y el desarrollo de páginas web para torneos de League of Legends. Estas actividades reflejan mi creatividad y mi interés por aplicar la programación en diversos contextos.
